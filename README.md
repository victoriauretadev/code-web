💻 Code Web - Landing Page

Landing page desarrollada para ofrecer servicios de diseño y desarrollo web, enfocada en atraer clientes y generar contacto directo a través de WhatsApp.

🌐 Demo del proyecto

👉 https://code-web-orpin.vercel.app/

🎯 Objetivo

El objetivo de este sitio es presentar servicios digitales de forma clara, moderna y persuasiva, optimizando la conversión de visitantes en clientes.

🚀 Tecnologías utilizadas
HTML5
CSS3
Diseño responsive (Mobile First)
Integración con WhatsApp
✨ Características
Diseño 100% personalizado (sin plantillas)
Estructura pensada para conversión
Secciones de servicios y beneficios
Botón directo a contacto por WhatsApp
Estilos modernos y atractivos
